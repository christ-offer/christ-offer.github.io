    About this site:

    What started out as a playground for postgrest + htmx, suddenly became my new personal website. The frontend is purely HTML/CSS (with htmx for any HTTP requests). The entire backend runs on Supabase, with simple functions to return HTML directly from the database

    Other than the aforementioned, the site is a simple blog, and a place for me to put some of my photographs. There is also a page where I show of projects I've worked on - as one does on one of these websites.
